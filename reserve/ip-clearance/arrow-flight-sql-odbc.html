    flightsql-odbc Intellectual Property (IP) Clearance Status

    Description

    flightsql-odbc is a library to help implement an ODBC driver that supports databases using the Apache Arrow Arrow Flight SQL protocol. It is not a complete ODBC driver on its own.

    Project info

    • The Arrow PMC will be responsible for the code
    • It will be integrated into the Apache Arrow project.
    • Officer or member managing donation: David Li/lidavidm (Apache Arrow PMC member)
