    Spark Connect Rust Codebase Intellectual Property (IP) Clearance Status

    Description

    The Spark Connect Rust project is a native Rust client implementation of the Apache Spark Connect protocol. It enables Rust applications to build DataFrame and Dataset queries and execute them remotely on Apache Spark clusters.

    Project info

    • Receiving PMC: Apache Spark
    • Officer / Member managing donation: Yuanjian Li
    • Source codebase: https://github.com/sjrusso8/spark-connect-rs
    • Target repository: https://github.com/apache/spark-connect-rust
    • Spark PMC acceptance vote: https://lists.apache.org/thread/mj3ggbsqhjr2lblffmyn1rq5zcfckcrn
