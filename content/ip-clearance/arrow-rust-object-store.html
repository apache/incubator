    Apache Arrow Rust Object Store Codebase Intellectual Property (IP) Clearance Status

    Description

    Arrow Rust Object Store provides a generic way to access local file systems and remote object storage systems such as AWS S3, Azure Blob Storage and Google Cloud storage, facilitating implementations of analytic systems that work with data stored on such systems. It also provides a generic interface to support additional similar systems.

    Project info

    • The Apache Arrow PMC will be responsible for the code
    • It will be integrated into the Apache Arrow Rust project arrow-rs
    • Officer or member managing donation: Andrew Lamb, (Apache Arrow PMC Member)

    Completed tasks are shown by the completion date (YYYY-MM-dd).
