    ServiceComb Toolkit Codebase Intellectual Property (IP) Clearance Status

    Description

    The ServiceComb Toolkit project is a microservice development tool to help users quickly build microservices projects, it supports generating microservice project which is based on some popular microservices frameworks (such as ServiceComb and SpringCloud), programming models (such as SpringMVC, POJO and JAX-RS) or Springbootï¼Œ and supports automatic extraction of contracts, contract verification, automatic generation of code and documentation.

    • It will be integrated with ServiceComb project as a submodule
