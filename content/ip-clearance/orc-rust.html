    ORC-rust Codebase Intellectual Property (IP) Clearance Status

    Description

    ORC-rust is the rust implementation to the Apache ORC format. It is being donated as a new repository within the Apache ORC project and contains a set of usable APIs for reading ORC format at present.

    Project info

    • The Apache ORC PMC will be responsible for the code
    • It will be integrated into the Apache ORC project in a new repository
    • Officer or member managing donation: Ruihang Xia (Apache DataFusion PMC member)
