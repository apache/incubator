    Apache Arrow PyArrow Stubs Codebase Intellectual Property (IP) Clearance Status

    Description

    pyarrow-stubs provides PEP 484 type stub annotations for PyArrow, the Python bindings for Apache Arrow. The stubs enable static type checking of PyArrow-based code using tools such as mypy and pyright. The donated codebase is the existing external library maintained at zen-xu/pyarrow-stubs, contributed in modified form as a series of pull requests into the main Apache Arrow repository.

    Project info

    • The Apache Arrow PMC will be responsible for the code.
    • It will be integrated into the Apache Arrow project (apache/arrow).
    • Officer or member managing donation: Rok Mihevc (Apache Arrow PMC Member)
