Apache Paimon (Incubating)

Paimon is a unified lake storage to build dynamic tables for both stream and batch processing with big data compute engines, supporting high-speed data ingestion and real-time data query.

  • Resources: paimon

  • Started: 2023-03-12; Last Status Update: 2023-12-14

  • Reporting: February, May, August, November

  • Committers: 14

  • PPMC members: 10

  • Mentors: Becket Qin (jqin), Robert Metzger (rmetzger), Stephan Ewen (sewen), Yu Li (liyu)
